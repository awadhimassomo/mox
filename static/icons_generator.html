Mo-Express Icon Generator

Instructions:

This tool will help you create placeholder icons for your Mo-Express PWA. Follow these steps:

  1. Draw your logo on each canvas below (or click "Fill with Placeholder" for a simple placeholder)
  2. Click the "Download" button below each canvas to save the icon
  3. Save the images to the following paths:
    • 192x192 icon → /static/icons/icon-192x192.png
    • 512x512 icon → /static/icons/icon-512x512.png
    • Favicon → /static/favicon.ico (Use the 192x192 icon for this)
